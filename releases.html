RynthCore

View on GitHub

The core x86 NativeAOT injection framework and launcher for Asheron's Call modding.

RynthCore v0.3

Latest Release
Released Apr 18, 2026

Installation:

  1. Download RynthCore-Setup.exe below and run it
  2. Start Asheron's Call and log in to your character
  3. Open RynthCore from the Start Menu or Desktop shortcut
  4. Click Inject Running AC in the launcher
  5. The RynthAi panel will appear inside the AC window

First-time Setup:

  • In the RynthCore launcher, set your acclient.exe path under Runtime Paths if prompted
  • Plugins auto-load from C:\Games\RynthCore\Runtime\Plugins\
  • Loot, nav, and meta profiles go in C:\Games\RynthSuite\RynthAi\
  • Log file: %USERPROFILE%\Desktop\RynthCore.log

System Requirements

RynthCore

  • OS: Windows 10/11 (x86/x64)
  • .NET: .NET 9 Runtime
  • AC Client: Latest Asheron's Call
  • RAM: 2GB minimum
  • Storage: 500MB free space

RynthSuite (Building)

  • OS: Windows 10/11
  • .NET SDK: .NET 9 SDK (x86)
  • IDE: Visual Studio 2022 (recommended)
  • Workloads: .NET desktop, C++ desktop
  • AC Path: Standard installation
