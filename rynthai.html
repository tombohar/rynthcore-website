RYNTHAI ENGINE

A high-performance intelligence suite for Asheron's Call, featuring advanced raycasting, state-based automation, and embedded Lua scripting.

Geometry Aware LOS

Beyond simple coordinate checks. RynthAi parses cell.dat Environment files to build a 3D mesh of dungeon walls and world geometry, providing true Line-of-Sight verification for targeting and movement.

Multi-Mode Combat

Seamlessly switch between Melee, Missile, and Magic. Includes intelligent weakness detection (e.g. Olthoi vs Cold), automated debuff rotations, and missile bundle crafting.

Meta Manager

The brain of the macro. Supports complex state machines with watchdog timers, state stacking (Call/Return), and native expression evaluation for ultra-precise automation triggers.

Tiered Navigation

Supports legacy SetAutorun, Tier 1 direct CM_Movement calls, and a planned Tier 2 MoveToPosition system for high-fidelity path following.

Lua Bridge

Extend RynthAi with standard Lua. Access player coordinates, world objects, and navigation functions through an embedded script engine for custom, per-character logic.

Smart Looting

Advanced property-based filtering with support for VTank .utl profiles. Includes auto-stacking, auto-cramming, and intelligent salvage combining to keep your packs clean.

Dungeon Pathfinding & LOS

RynthAi features a custom binary parser for AC cell.dat files. It extracts CellStruct data, transforms polygon vertices based on landblock positions, and generates BoundingVolume meshes for real-time raycasting.

  • CVertexArray Processing: Raw binary parsing of SWVertex and Type-1 vertex arrays.
  • Environmental Geometry: Support for 0x0Dxxxx EnvironmentId structures and portals.
  • Scatter System: High-performance spatial indexing for 100+ concurrent geometry checks.

Advanced Target Logic

The Combat Manager doesn't just click targets; it orchestrates a full battle sequence. It tracks debuff confirmations via chat logs to prevent re-casting and uses a TargetingFSM to handle switching between targets efficiently.

  • Blacklist Management: Automatic blacklisting of unreachable or LOS-blocked mobs (5s vs 20s timeouts).
  • Buff Synchronization: Real-time tracking of enchantments with support for Incantation of X Self lookups.
  • Stance Control: Automated peace/combat state switching based on proximity and idle timers.
