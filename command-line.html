COMMAND LINE

Control RynthAi and the RynthCore host directly from the game chat using the /na prefix.

Core Controls

/na start | on Starts the macro engine and enables all active subsystems.
/na stop | off Stops the macro engine, clearing navigation and combat targets.
/na status Prints current engine status, state, and loaded profiles.
/na ui Toggles the RynthAi ImGui overlay visibility.

Navigation & Movement

/na nav load <file> Loads a VTank-compatible .nav route file.
/na addnavpt Adds a waypoint at your current location to the active route.
/na reverseroute <true|false> Reverses the direction of waypoint traversal.
/na reverseroutequery Displays current route reversal status.
/na jump[swzxc] <heading> [holdtime] Executes jump with direction keys: s=shift, w=forward, z=left, x=back, c=right.
/na tapjump Performs a single jump action.
/na face <heading> Immediately turns your character to the specified heading (0-360).

Buffing & Combat

/na rebuff | forcebuff Initiates a complete buff sequence on yourself.
/na cancelforcebuff Cancels an active buff sequence.
/na setattackbar <melee|missile> <0-1> Sets melee or missile attack power (0.0 = 0%, 1.0 = 100%).
/na autoattackpower <auto|0-100> Sets both melee and missile power, or 'auto' for full power.
/na blacklist [clear] View blacklist settings or clear the blacklist.

Diagnostics & Testing

/na lostest Runs line-of-sight test diagnostics against current target.
/na raycast Shows raycast system initialization status.
/na probe Displays CombatActionHelper pattern scan status.
/na testattack Sends a single melee attack to current selection (combat disabled).
/na spelldump <keyword> Searches spell database for spells matching a keyword.
/na bufftest <spell> Traces the complete buff lookup path for a spell name.
/na debugbuffs | db Prints active buff tracking debug info.
/na debugnav | dn Toggles navigation system debug output.

Meta System

/na meta load <file> Loads a VTank-compatible .met meta profile.
/na setmetastate <state> Forces meta engine to transition to a specific named state.
/na watchdog | wd Prints the status of the active state watchdog timer.
/na fakedeath Simulates a character death event for testing recovery logic.

Fellowship & Corpses

/na fellow status Shows detailed fellowship info, member list, and roles.
/na fellow leader Displays current fellowship leader name and ID.
/na fellow count Shows total number of fellowship members.
/na fellow names Lists all fellowship member names.
/na fellow ismember <name> Checks if a specific player is in the fellowship.
/na fellow [open|locked|sharexp|name] Queries various fellowship properties.
/na corpsedebug Shows detailed ownership and killer info for selected corpse.

Looting & Inventory

/na loot load <file> Loads a VTank-compatible .utl loot profile.
/na opt set enablelooting <true|false> Enables or disables the looting subsystem.

Settings & Configuration

/na settings save Saves current settings to disk.
/na opt set <option> <value> Sets configuration options (enablecombat, enablenav, etc).
/na echo <text> Echoes text to chat for debugging or notifications.

Scripting

/lua <script> Executes a raw Lua string through the embedded engine.
/na lua <script> Alternative syntax for executing Lua scripts.
/na mexec <expression> Executes a VTank/UB mexec expression.
