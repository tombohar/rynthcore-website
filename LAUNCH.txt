🎯 QUICK START (5 MINUTES)
════════════════════════════════════════════════════════════════════════════

1. CREATE GITHUB REPO
   - Go to github.com/new
   - Name: "rynthcore-website"
   - Make it PUBLIC
   - Click "Create repository"

2. CLONE & PUSH
   git clone https://github.com/YOUR_USERNAME/rynthcore-website.git
   cd rynthcore-website
   cp /c/projects/website/nexcore/*.html .
   git add .
   git commit -m "Launch website"
   git push origin main

3. ENABLE PAGES
   - Go to Settings → Pages
   - Source: Deploy from branch
   - Select: main / (root)
   - Click Save

4. LIVE! 🎉
   Your site goes live at:
   https://YOUR_USERNAME.github.io/rynthcore-website/
